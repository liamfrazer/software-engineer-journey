# Revisiting Variables and Values

* String
* Text values
* Wrapped with single or double quotes
* Can also be created with backticks
```js
"Hello World"
'Liam'
`Hello there`
```

* Number
* Positive or Negative
* With decimal points (float) or without (integer)
```js
5
-23

3.14
-8.12
```

* Boolean
* true or false
* A simple "yes" or "no" value type
